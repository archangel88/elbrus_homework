Общие сведения

HTML (HyperText Markup Language) — стандартный язык разметки документов во Всемирной паутине.

Разработан для структурирования текста, изображений, таблиц, ссылок и других объектов, отображаемых в браузере.

Документы HTML — это обычные текстовые файлы с расширением .html или .htm.

Пример кода: <p>Привет, мир!</p>

История

HTML был предложен Тимом Бернерс-Ли в 1991 году как способ обмена научной информацией. [1]

HTML стал основой для развития веб-технологий.

С 1995 по 2014 год HTML прошёл путь от версии 2.0 до HTML5.
